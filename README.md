# Floormate

Калькулятор раскладки напольного покрытия: рассчитывает схему укладки панелей по размерам комнаты и визуализирует результат на canvas.

**Сайт:** https://jrosn.github.io/floormate/

## Возможности

- Расчёт раскладки с учётом отступа от стены и минимальной длины обрезка
- Смещение швов: 1/2, 1/3 панели или хаотичное (с переиспользованием обрезков)
- Визуализация схемы и экспорт в PNG
- Сохранение параметров в браузере

## Запуск локально

```bash
./gradlew jsBrowserDevelopmentRun
```

Сборка production-версии:

```bash
./gradlew browserDistribution
```

Артефакты появятся в `build/dist/js/productionExecutable`.

## Стек

Kotlin/JS, Gradle, Canvas API
